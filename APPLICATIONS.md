# Applications (App IDs)

Samsung TVs identify apps by an **application ID**.  
To launch, install, close, or query apps via `samsungtvws`, you typically need the correct app ID for your TV model / firmware.

> IDs can change over time and can differ by TV year / Tizen version.

## Finding App IDs via Samsung Website (HTML Inspect)

You can extract app IDs from Samsung’s Smart Hub / app listing pages:

1. Open Samsung’s app listing page in your browser (example: Smart Hub / TV apps pages).
2. Right-click an app icon and select **Inspect**.
3. Locate the `<button>` element related to the app.
4. Read the `data-app-id` attribute (this is the application ID).

> Tip (Safari)  
> Enable web inspector: **Safari Settings → Advanced → “Show features for web developers”**.

## Getting Installed Applications from Your TV (Tizen Studio)

> Warning  
> Samsung removed the API previously used to fetch installed apps directly from the plugin.  
> The remaining option is more involved and requires **Tizen Studio**.

### Steps

1. Install **Tizen Studio** and **TV Extensions**

   - Install guide:  
     https://developer.samsung.com/smarttv/develop/getting-started/setting-up-sdk/installing-tv-sdk.html

2. Enable **Developer Mode** on your TV and set the **PC IP** (where Tizen Studio runs)

   - TV device setup guide:  
     https://developer.samsung.com/smarttv/develop/getting-started/using-sdk/tv-device.html

3. Open **Device Manager** in Tizen Studio

   - Right-click your TV device entry
   - Select **Installed Web App List**
   - If the option is missing, confirm TV Extensions were installed.

4. A list of installed apps appears

   - Use the value from **`App ID - WAS`** as the app ID.

5. You can uninstall Tizen Studio after extracting the IDs.
